Interactive dashboard to explore specific airports¶

I am a huge proponent of interactive data visualization because I believe it can lead you to data-driven insights faster while also remaining more intuitive to a user.

Here I build a Dash app that can be used to select certain points in the relationship and filter down the datatable below based on what is selected. This is a great repeatable pattern that enables faster iteration of ideas, and in the limited time I just made this very simple. With more time I would polish this to have other inputs/selectors, etc. (as well as take it out of jupyter lab).

In [15]:
import dash
from dash import dcc, html, dash_table
from dash.dependencies import Input, Output
import plotly.express as px
import pandas as pd

# Filter DataFrame
data_df = df[df['year_qt'] == '2024-Q3'].copy()

app = dash.Dash(__name__)

# Layout
app.layout = html.Div([
    dcc.Graph(
        id='scatter-plot',
        figure=px.scatter(
            data_df, x='TotalAvgHubFare', y='TotalFaredPax', custom_data=['tbl7pk']
        ),
        config={'editable': False}
    ),
    dash_table.DataTable(
        id='data-table',
        columns=[{"name": i, "id": i} for i in data_df.columns],
        data=data_df.to_dict('records'),
        row_selectable="multi"
    ),
    html.Div(id='selected-data')
])

# This will update the table based on the data selected from the callback
# It's extra helpful that we have the primary key available.
@app.callback(
    Output('data-table', 'data'),
    Input('scatter-plot', 'selectedData')
)
def update_table(selectedData):
    if selectedData and 'points' in selectedData:
        selected_labels = [point['customdata'][0] for point in selectedData['points']]
        filtered_df = data_df[data_df['tbl7pk'].isin(selected_labels)]
        return filtered_df.to_dict('records')
    else:
        return data_df.to_dict('records')

app.run()

Seasonality Decomposition¶

With the remaining time, I thought it would be interesting to see if there was an overall seasonality trent present quarter over quarter across a few years. For this I defined a repeatable function that would allow you to plot the multiple time series decompositions.

In [16]:
import pandas as pd
import numpy as np
from statsmodels.tsa.seasonal import seasonal_decompose, STL , MSTL
import plotly.graph_objects as go
from plotly.subplots import make_subplots

def plot_components(result):

  df = pd.concat([result.observed, result.trend, result.seasonal, result.resid], axis=1)
  df = df.rename(columns={0:'Original Data', 'season':'Seasonal','observed':'Original Data'})
  components = df.columns
  rows = len(components)
  fig = make_subplots(rows=rows, cols=1, shared_xaxes=True, subplot_titles = [i for i in components])
  
# Plot original data
  for i, col in enumerate(components):
    fig.add_trace(go.Scatter(x=df.index, y=df[col], mode='lines', name=col), row=i+1, col=1)


  # Update layout
  fig.update_layout(
      title='Time Series Decomposition',
      xaxis_title='Time',
      height=1200,
      width=800
  )

  fig.show()

After this, I utilize the function after modeling this after a particular set of complete cycle of 3 years (requires Q1-Q4 for all years), for a single airport.
